What's Does it mean to display inline vs inline blocks?

Sometimes we want to make our website look pretty by highlighting certain parts of text in a sentence as shown here. This is done by using the "display inline" In your CSS file. An example of this in actioan and the code used on this website looks like this.

So lets break down whats going on in these screenshots shall we. The image on the left is the class created in the css file which determines four things here:

  1. Padding: The amount of space between the outside of the box and the text insude.
  2. Border: The traits of the border including what type how thick and the color
  3. Background color: Determines the background color of the highlighted section in this case its Yellow
  4. Display: Determines how the highlighted section will be displayed in this case we used inline.

The image on the right is in our html file and is how we use the code in action. The important part here is the line < span > class="inline">highlighting < /span > This will use the class "inline" created in the css file to apply to the text highlighting. The constraints of using inline to highlight text is that we are unable to set the dimensions of the section to be any bigger or smaller based on our needs. If we want to determine the width and height of the box we can use inline-block as shown below.

The code used to make these cyan highlighted sections are almost identical to display inline with one key difference being that there are now width and height paramaters that can be set. Another use for inline-block is to display lists horizontally reather than vertically as inline block does not create a line braek after use.
